# 📉 Customer Churn Prediction

## 🚀 Project Overview

Customer Churn Prediction is a Machine Learning project developed using **Google Colab** to identify customers who are likely to leave a company or discontinue a service. By analyzing customer behavior and historical data, the model predicts churn and helps businesses improve customer retention strategies.

This project demonstrates the complete Machine Learning workflow including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

---

## 🎯 Objectives

- Analyze customer data and identify churn patterns.
- Preprocess and clean the dataset.
- Train Machine Learning models for churn prediction.
- Evaluate model performance using classification metrics.
- Predict whether a customer is likely to churn or stay.

## 📊 Dataset Features

The dataset may contain the following customer information:

- Customer ID
- Gender
- Age
- Tenure
- Contract Type
- Monthly Charges
- Total Charges
- Internet Service
- Payment Method
- Customer Support Calls
- Churn Status

**Target Variable:**
- Churn (Yes / No)

## 🔍 Project Workflow

### 1️⃣ Data Collection
- Import customer churn dataset into Google Colab.

### 2️⃣ Data Preprocessing
- Handle missing values.
- Encode categorical variables.
- Feature scaling and transformation.

### 3️⃣ Exploratory Data Analysis (EDA)
- Analyze customer demographics.
- Visualize churn distribution.
- Study feature correlations.

### 4️⃣ Model Building
Machine Learning algorithms used:

- Logistic Regression
- Random Forest Classifier
- Decision Tree Classifier

### 5️⃣ Model Evaluation
Evaluate performance using:

- Accuracy Score
- Precision
- Recall
- F1 Score
- Confusion Matrix

### 6️⃣ Prediction
Predict whether a customer will churn or remain with the company.
